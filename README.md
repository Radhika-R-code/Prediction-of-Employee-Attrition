# Employee Attrition Prediction

This project is a machine learning-based web application built using Streamlit to predict employee attrition risk. It provides workforce insights through dashboards and enables both single employee and batch attrition predictions.

# Key Features
- Workforce overview dashboard with attrition metrics
- Attrition distribution and department-level insights
- Single employee attrition prediction
- Batch prediction using CSV upload
- Downloadable prediction results

# Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Joblib

# How to Run
Install dependencies and run:
streamlit run app.py
